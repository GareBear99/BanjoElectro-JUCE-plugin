# Obsidian Electric String Pack

This rebuild pushes the previous dark banjo set toward a haunted picked electric-string / electric-guitar-adjacent tone.

## What changed
- less banjo body ring
- longer sustain
- stronger upper-mid pickup bite
- darker magnetic-string tone
- same ominous harmonic writing, but with more electric edge

## Included
### Loops
- Obsidian Hollows Electric Arp
- Cathedral Electric Tremor
- Ritual Ash Electric Pulse
- Veilglass Electric Harmonics
- Midnight Electric Triptych
- Black Marrow Electric Picking

### One-shots
- tuned electric-string plucks for layering, resampling, and custom riffs

### Script
- Python synth script used to generate these tones

## Technical
- 48 kHz
- 32-bit float WAV
- original synthesized material
